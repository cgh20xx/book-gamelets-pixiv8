# book-gamelets

與《Pixi.js讓新手也能寫出好遊戲》配合的遊戲專案 (使用pixi.js v8)

# 快速開始

```
// 安裝需要的套件
npm install

// 啟動測試伺服器
npm run dev
```

伺服器啟動後，在瀏覽器上打開 http://localhost:5173/ 就可以看到專案成品。
